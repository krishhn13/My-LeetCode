## üìÅ Repository Structure

The repo is divided into two main directories:

- **Serial OrderWise/**: Solutions grouped by LeetCode problem ID (e.g., `LeetCode 01 - Two Sum`). Each folder contains one or more `.java` files, often including brute-force and optimized implementations.
- **Topic Wise/**: Solutions categorized by algorithm topics (e.g., `Greedy`, `Linked List`). This helps with targeted practice.

Additional files:
- `LICENSE`: MIT License for open usage.
- `.github/workflows/`: CI/CD setups (e.g., automated README updates).

## üöÄ How to Use

1. **Clone the Repo**:
   ```
   git clone https://github.com/krishhn13/My-LeetCode.git
   cd My-LeetCode
   ```

2. **Run a Solution** (e.g., Two Sum):
   - Navigate to `Serial OrderWise/LeetCode 01 - Two Sum/`.
   - Compile: `javac Optimised.java`
   - Run: `java Solution` (add your main method or use an IDE like IntelliJ/VS Code for testing).

3. **IDE Recommendations**: Use IntelliJ IDEA, Eclipse, or VS Code with Java extensions for easy debugging.

4. **Testing**: Each solution includes the standard LeetCode `Solution` class. Test with sample inputs from the problem statements.
